## ğŸš€ Quick Start

### Prerequisites

1. **Install Quarto**: Download from [quarto.org](https://quarto.org/docs/get-started/)
2. Verify installation:
   ```bash
   quarto --version
   ```
3. **Render the Website**
   ```bash
   quarto render
   ```
   This generates the website in the `docs/` folder

4. **Preview the Website**
   ```bash
   quarto preview
   ```
