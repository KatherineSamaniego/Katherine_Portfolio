# Assets Folder

This folder contains images, PDFs, and other media files for the portfolio website.

## Required Files

Please add the following files to this folder:

### 1. Profile Photo
- **Filename**: `profile-photo.jpg`
- **Recommended size**: 500x500px (square)
- **Format**: JPG or PNG
- **Description**: Professional headshot for the home page

### 2. Resume PDF
- **Filename**: `Katherine_Samaniego_Resume.pdf`
- **Format**: PDF
- **Description**: Your complete resume in PDF format

### 3. Optional: Project Images
- Add any screenshots, mockups, or visualizations from your MBA projects
- Examples:
  - `el-paisano-mockup.png` - E-commerce website mockup
  - `netflix-report-chart.png` - Charts from your Netflix AI report
  - `leadership-presentation.png` - Slide from your leadership presentation

## File Naming Best Practices

- Use lowercase letters
- Replace spaces with hyphens (e.g., `my-project-image.jpg`)
- Use descriptive names
- Keep filenames short and simple

## Image Guidelines

- **Max file size**: 2MB per image (compress large images)
- **Formats**: JPG for photos, PNG for graphics with transparency
- **Accessibility**: All images should have descriptive alt text in the `.qmd` files
